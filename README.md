## 🔧 문제 및 해결사항

| 문제 | 해결 방안 |
|------|-----------|
| 프론트에서 CORS 오류 발생 | Spring Boot `@CrossOrigin` 또는 `WebMvcConfigurer`로 허용 설정 |
