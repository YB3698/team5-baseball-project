> **개발 기간**: 2025.06.13 ~ 2025.06.25  
> **팀 구성**: 김정현, 양은지, 황혜미, 박유빈

---

## 📌 프로젝트 개요

“야~! 모여”는 야구 팬들을 위한 통계 기반 커뮤니티 플랫폼입니다.  
KBO 데이터를 활용한 선수/팀 통계 제공, 팬 투표, 응원팀 게시판, AI 챗봇, 실시간 채팅 등 다양한 기능을 통해 팬들 간 소통과 재미를 제공합니다.

---

| Crawling | Python, BeautifulSoup, pandas |

- `GlobalExceptionHandler` 구성
- 400, 403, 404, 500 등 표준 에러 코드 및 메시지 적용
- 프론트에서 에러 메시지 모달 또는 Toast로 출력

- React 컴포넌트 UI 구성
